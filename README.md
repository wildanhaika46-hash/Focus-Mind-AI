# FocusMind AI

> **Chrome Extension AI untuk Fokus, Belajar, dan Produktivitas**

---

## 1. Deskripsi Proyek & Masalah yang Diselesaikan

FocusMind AI adalah ekstensi Chrome profesional berbasis Gemini Nano (Chrome Built-in AI) untuk membantu pelajar dan profesional:
- Mengurangi distraksi digital (blokir TikTok, IG, Shorts)
- Memahami materi online lebih cepat (AI Summarizer)
- Menulis catatan belajar efisien (Quick Notes, Rewriter, Proofreader)
- Meningkatkan produktivitas harian (Focus Mode & Analytics)

---

## 2. API yang Digunakan & Alasan
- **Prompt API**: Motivasi harian, instruksi AI (real-time, relevan)
- **Summarizer API**: Meringkas materi online (cepat, bullet points)
- **Rewriter API**: Menyederhanakan paragraf (mudah dipahami)
- **Proofreader API**: Koreksi grammar & ejaan (instan, akurat)
- **Translator API**: Terjemahan ringkasan (multi-bahasa)

> Semua API ini native, cepat, dan aman (tidak kirim data ke server eksternal).

---

## 3. Cara Instalasi & Menjalankan Ekstensi

1. **Clone/download** repo ini
2. Buka **Chrome > Extensions > Developer Mode**
3. Pilih **Load unpacked** > arahkan ke folder `FocusMindAI`
4. Klik ikon 🧠 di toolbar Chrome untuk mulai

---

## 4. Fitur Utama & Cara Uji

- **Focus Mode**: Blokir situs gangguan, timer fokus, motivasi AI (uji: klik ⏱ Focus Mode)
- **AI Summarizer**: Ringkas materi tab aktif (uji: klik 📚 Summarizer)
- **Smart Rewriter**: Sederhanakan paragraf (uji: klik ✍️ Rewriter)
- **Proofreader**: Koreksi catatan (uji: klik 📝 Proofreader)
- **Translator**: Terjemahan instan (uji: klik 🌐 Translator)
- **Quick Notes**: Simpan/edit/hapus catatan (uji: klik 🗒 Quick Notes)
- **Productivity Analytics**: Grafik waktu fokus (uji: klik 📈 Analytics)
